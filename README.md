# Clothes Shop Website

A fully responsive e-commerce website for a clothing store. Built with **HTML**, **CSS**, and **JavaScript**.  
This project showcases product listings, dynamic search, shopping cart functionality, and breadcrumb navigation.

---

## Features

- **Product Pages**: Individual product pages with detailed info, images, and pricing.
- **Dynamic Breadcrumbs**: Navigate easily through the site and track visited pages.
- **Shopping Cart**: Add, remove, and update product quantities. Cart totals and discounts are dynamically calculated.
- **Search Functionality**: Search products by title across all sections including New Arrivals, On Sale, Men, Women, and Shop pages.
- **Responsive Layout**: Works well on desktop, tablet, and mobile.
- **Discount Handling**: Displays original prices, discounts, and final prices clearly.
- **Light/Dark Mode Toggle**: Switch between light and dark themes for better user experience
